# VR Evacuation Experiment — Analysis Code Repository

## Important Notice for Reviewers

This repository has been prepared for the purpose of paper submission and peer review. Please note the following:

1. **Code-Only Repository**: This repository contains only data analysis source code (Python scripts). No experimental raw data, processed data, intermediate results, or final result files are included.
2. **Privacy Protection**: All privacy-sensitive data, specifically participant names, has been anonymized via MD5 hashing prior to any code being committed to this repository. No personally identifiable information is present in any file within this repository.

## About This Project

This repository provides the complete analysis codebase for a virtual reality evacuation experiment investigating how leader presence (Passenger / Robot / Security) influences participant evacuation decision-making and behavior. The code covers data aggregation, metric extraction, statistical testing, Bayesian hierarchical modeling, mixed-effects logistic regression, DBSCAN-based eye-gaze clustering, DTW trajectory similarity analysis, and various visualization routines.
